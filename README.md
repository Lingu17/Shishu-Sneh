# 🚀 Shishu-Sneh – Smart Baby Care & Growth Tracking App

Shishu-Sneh is a smart baby care and healthcare management application designed to help parents monitor their child’s growth, health records, feeding schedules, vaccination reminders, and daily activities efficiently.

The application provides a centralized and user-friendly platform for managing baby healthcare and development tracking.

---

# 🌟 Features

- 👶 Create and manage baby profiles
- 📈 Track baby growth and health records
- 💉 Vaccination reminder system
- 🍼 Feeding schedule management
- 💤 Sleep and activity tracking
- 🔔 Smart notifications and reminders
- 📊 Milestone and development tracking
- 🎨 Clean and user-friendly Android interface

---

# 🧠 How It Works

1. Parent registers and logs into the application
2. Baby profile details are added
3. Health records and schedules are stored
4. The system tracks growth and milestones
5. Notifications remind users about important activities

# 🎯 Problem Statement

Many parents find it difficult to manage baby healthcare records, feeding schedules, vaccination reminders, and growth tracking manually. Missing important schedules can affect child healthcare management.

Shishu-Sneh solves this problem by providing a centralized smart baby care system that helps parents efficiently monitor and manage their child’s healthcare activities.
